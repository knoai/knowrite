# 小说创作核心要素 —— Prompt 优化工程手册

> 本手册为 knowrite 小说引擎的 Prompt 设计提供标准化指引。所有核心要素均已转化为可直接注入 LLM 的指令模板。

---

## 一、十二项核心要素（含 Prompt 模板）

### 1. 每章钩子（Hook）
**定义**：在章节开头 200 字内建立"不得不读下去"的引力，结尾 200 字内留下强烈期待。

**Prompt 模板**：
```
本章开头必须在 200 字内出现以下至少一种元素：冲突爆发、反常事件、核心悬念的碎片、主角面临的关键抉择。禁止大段景物描写和背景铺陈。
本章结尾必须留下一个强钩子：未解的悬念、即将到来的对决、意味深长的台词、或主角命运的重大转折。确保读者有强烈的"点下一章"冲动。
```

**正例**：
- ✅ "棺材板被掀开的瞬间，他看到了自己的脸。"
- ✅ "如果这一剑斩下去，她就会死——可若不斩，死的就是整座城。"

**反例**：
- ❌ "那是一个风和日丽的早晨，小鸟在枝头歌唱……"（无冲突、无悬念）
- ❌ 结尾收束过于圆满，没有任何未解问题。

---

### 2. 悬念（Suspense）
**定义**：利用信息差制造张力。分为全书核心悬念（贯穿始终的终极问题）和章节悬念（单章内建立并部分释放）。

**Prompt 模板**：
```
建立悬念时必须遵循以下原则：
1. 信息差要清晰——读者知道主角不知道，或主角知道读者不知道，或双方都不知道但危险正在逼近；
2. 每个悬念在揭晓前必须有 2-3 处逻辑铺垫（伏笔），禁止毫无征兆的强行反转；
3. 核心悬念（全书终极问题）必须在本章有所呼应或推进，哪怕只是透露一个新的碎片；
4. 单章悬念尽量在本章末或下章初给出阶段性回应，避免长期悬而不决导致读者疲惫。
```

**检查点**：
- 这个悬念是否让读者产生了"为什么/会怎样"的追问？
- 揭晓时是否能让读者发出"原来如此"而不是"这什么鬼"的感叹？

---

### 3. 视觉冲击（Visual Impact）
**定义**：用文字构建可直接在脑中成像的画面，减少抽象形容词，增加具象名词和动态动词。

**Prompt 模板**：
```
所有场景描写必须遵循"具象化原则"：
- 禁止大量使用"非常"、"很"、"仿佛"、"似乎"等模糊词；
- 每个画面至少包含 1 个具体细节（如"青铜灯盏上的裂纹"而非"古老的灯"）；
- 动作描写优先使用精准动词（如"砸"、"碾"、"撕"而非"用力打"）；
- 重要场景必须通过五感中的至少两种来呈现（视觉 + 听觉/触觉/嗅觉）。
```

**改写示例**：
- ❌ 原句："战斗非常激烈。"
- ✅ 改写："剑锋撞上骨刃，火星溅在他睫毛上，烫出一股焦糊味。"

---

### 4. 情感冲击（Emotional Impact）
**定义**：通过细节让读者与角色产生共振，而不是直接告诉读者"他很悲伤"。

**Prompt 模板**：
```
情感表达必须"藏"在细节里：
- 悲伤：不要写"他很难过"，要写他反复摩挲一只旧茶杯，指腹沿着缺口转了三圈；
- 愤怒：不要写"他很生气"，要写他把筷子捏断，断茬扎进掌心却浑然不觉；
- 恐惧：不要写"他很害怕"，要写他数着楼梯阶数，发现多出了一阶；
- 喜悦：不要写"他很高兴"，要写他走了三条街才发现自己忘了穿鞋。
严禁强行卖惨、情绪脱节、或为了虐而虐。
```

**反差感技巧**：
> 在极端情境下写日常小动作（生死关头整理衣领），或在日常情境下写极端情绪（平静湖面下汹涌的杀意）。

---

### 5. 代入感（Immersion）
**定义**：打破第四面墙，让读者感觉自己就是主角，或正站在主角身边。

**Prompt 模板**：
```
增强代入感的四个技术：
1. 感官同步：主角感受到的疼痛、冷热、窒息感，必须精确传递给读者（如"胃酸涌到喉头"而非"他很难受"）；
2. 信息受限：读者的信息视野不应大于主角（悬疑/危机时刻），或恰好与主角同步（让读者替主角紧张）；
3. 普通人属性：主角必须有软肋——怕黑、怕高、怕辜负某人、或只是单纯怕疼；
4. 困境普遍化：把"拯救世界"降级为"保护眼前这一个具体的人"，宏大目标必须通过私人情感落地。
```

---

### 6. 人物缺陷（Character Flaw）
**定义**：缺陷是角色的锚点，也是成长弧光的起点。完美主角等于无聊主角。

**Prompt 模板**：
```
每个主要角色必须携带至少一个明显缺陷，且该缺陷必须真实影响情节走向：
- 性格缺陷：自负、怯懦、偏执、过度善良、不信任任何人；
- 能力缺陷：某项关键技能的缺失、身体上的旧伤、无法控制的副作用；
- 心理缺陷：创伤后遗症、执念、身份认同危机。
缺陷必须贴合角色的身份与经历（富家子弟的傲慢、孤儿的不安全感），禁止为缺陷而缺陷。
角色的高光时刻往往发生在"缺陷被克服"或"缺陷导致重大代价"之时。
```

---

### 7. 情绪价值（Emotional Value）
**定义**：读者阅读小说的根本动机——来这里获取他们在现实中缺失的情绪体验。

**Prompt 模板**：
```
本章必须提供至少一种明确的情绪价值，并保持情绪基调的连贯性：
- 宣泄型：压抑后的爆发、屈辱后的反击、失去后的复仇；
- 慰藉型：被理解、被拯救、在绝望中发现微光；
- 新奇型：探索未知世界的惊奇、掌握新能力的快感、破解谜题的智力愉悦；
- 共鸣型："这就是我曾经历过的"——成长阵痛、离别、遗憾、暗恋。
情绪转折必须有事件支撑，禁止无缘无故的情绪跳跃。避免连续多章过度负面情绪（致郁）或完全无脑爽（空洞）。
```

---

### 8. 小说意义（Theme / Meaning）
**定义**：故事的内核，是读者合上书后仍然记得的东西。可以宏大，也可以小而美。

**Prompt 模板**：
```
本小说的核心主题是：{theme}。
所有情节发展、人物选择、高潮设计都必须能回溯到这个主题。禁止为了炫技而设计脱离主题的情节。
主题表达必须通过"展示"而非"说教"完成：
- 不要说"友谊很珍贵"，要展示主角为了朋友放弃唯一活命的机会；
- 不要说"权力腐蚀人心"，要展示同一个人掌权前后的言行对比。
主题可以小而美（一个关于如何与过去和解的故事），也可以宏大（关于人类文明存续的思辨），但必须真诚、有内核、不空洞。
```

---

### 9. 节奏把控（Pacing）
**定义**：叙事速度的调控艺术。好的节奏让读者在"喘不过来气"和"意犹未尽"之间循环。

**Prompt 模板**：
```
本章节奏必须张弛有度，遵循以下波形原则：
- 高潮戏（战斗、对峙、揭秘）：短段落、高频动作、快节奏对话，每 300 字必须有一个小刺激点；
- 过渡戏（赶路、疗伤、日常）：可适当放缓，但必须埋设伏笔、推进人物关系、或展示世界规则；
- 禁止平铺直叙：即使是过渡戏，也要让角色之间产生张力（意见不合、隐藏秘密、暗流涌动）；
- 连续高潮后必须给 1 章喘息空间，连续压抑后必须给 1 章情绪释放。
```

**节奏检测公式**：
> 1 章内，对话 : 动作 : 描写 的比例建议控制在 **5:3:2**。描写超过 30% 容易拖沓。

---

### 10. 逻辑闭环（Logical Consistency）
**定义**：故事的自洽性。伏笔必须回收，规则一旦建立就不能随意打破。

**Prompt 模板**：
```
本章创作前，请先核对以下逻辑检查清单：
1. 人物行为是否符合其当前动机、性格和已知信息？
2. 本章节使用的任何能力/道具/规则，是否在前文已有铺垫或属于已建立的世界观体系？
3. 新揭晓的悬念是否有至少 2 处前文伏笔可以回溯？
4. 时间线、地理移动、人物关系是否与上一章一致？
5. 是否存在"天降神兵"式的强行解围？（若有，必须在 3 章内解释其来源合理性）
禁止为了推进剧情而让人物"降智"、"性情大变"或"强行巧合"。
```

---

### 11. 配角塑造（Supporting Characters）
**定义**：配角不是工具人，他们是主角的镜子、对手、催化剂。

**Prompt 模板**：
```
每个有台词的配角必须满足以下至少两项：
- 有独立立场和动机（即使是反派，也要相信自己做的事是对的）；
- 有性格缺陷或鲜明怪癖（口吃、洁癖、偏执的正义感、对某种颜色的狂热）；
- 有自己的成长或堕落轨迹（不只是主角在变，配角也在变）；
- 与主角有独特的互动模式（只有他们两个人才懂的默契、特殊的称呼方式、共同的禁忌话题）。
严禁以下行为：
- 用完即弃的工具人（出现只为送道具/传消息/领便当）；
- 抢戏的配角（篇幅超过主角且没有节制）；
- 浪费的设定（铺垫了很久的神秘强者，出场一句话就死了）。
```

---

### 12. 语言质感（Language Texture）
**定义**：文字本身的审美体验。好的语言是透明的——它不会阻碍读者进入故事，反而会加速沉浸。

**Prompt 模板**：
```
本章语言必须贴合 {style} 的整体调性，并遵循以下技术规范：
1. 句式错落：长句用于渲染氛围，短句用于制造紧张感，禁止连续三个以上结构相似的排比句；
2. 对话功能化：每一句对话必须同时承担至少两项功能（推进情节 + 展示性格 / 制造冲突 + 埋藏信息）；
3. 零废话原则：删除所有不服务于情节、人物或氛围的描写；
4. 去 AI 腔：禁用"不仅如此"、"从某种意义上说"、"不可否认的是"等通用套话；
5. 精准用词：一个场景里不要同时出现"非常"、"很"、"特别"超过 3 次。
```

## 二、knowrite 专项 Prompt 注入

### 针对编辑 Agent 的审查清单
```
请从以下 12 个维度审查本章稿件，并给出修改意见：
□ 钩子：开头/结尾是否有足够的吸引力？
□ 悬念：信息差是否清晰？铺垫是否充足？
□ 视觉：画面是否具象？有无抽象空洞的描写？
□ 情感：情绪是否通过细节传递？有无强行卖惨？
□ 代入：感官描写是否到位？读者能否与主角同步？
□ 缺陷：主角/配角是否有明显缺陷并影响情节？
□ 情绪价值：本章提供了哪种情绪体验？是否连贯？
□ 意义：情节是否服务于核心主题？有无说教？
□ 节奏：张弛有度？有无拖沓或仓促？
□ 逻辑：人物行为是否自洽？伏笔是否回收？
□ 配角：有无工具人/抢戏/浪费设定？
□ 语言：是否符合风格调性？有无 AI 腔/堆砌/废话？

请输出：【总体评价】+ 【问题清单（按严重程度排序）】+ 【修改建议】。
最后明确输出评审结论：【评审结果：通过】 或 【评审结果：不通过】。
```

### 针对作者改稿 Agent 的改稿指令
```
你是一位职业网络小说作家。请根据编辑意见对当前稿件进行修改，特别注意：
1. 优先修正编辑标记为"严重"的问题；
2. 在修改悬念/伏笔时，确保不破坏已有的逻辑链条；
3. 增强画面的具象化描写，删除抽象形容词；
4. 检查本章的"钩子"和"情绪价值"是否在改稿后更加突出；
5. 保持 {style} 的语言风格，禁止出现 AI 通用套话。

请输出修改后的完整章节，并在文末附上【修改说明】，列出你做了哪些关键调整及原因。
```

### 针对去 AI 化 Agent 的风格化指令
```
你的任务是去除文本中的"AI 腔"，并强化 {style} 的语言质感。请重点执行以下操作：
1. 句式：将连续排比改写为长短错落的句式；
2. 对话：让口语更贴合人物身份（莽夫要粗粝，谋士要藏锋，少年要热血）；
3. 修辞：把所有"非常/很/仿佛"替换为更具体的描述；
4. 细节：每 500 字至少加入 1 个只有人类作家才会注意到的微小细节；
5. 语感：朗读修改后的段落，如果读起来像公文/报告/论文，就继续改，直到像小说。
```

## 三、"三抓三建四检查"执行标准

### 三抓
- **抓情感**：每章必须有一个明确的情绪主调，且该情绪必须通过细节传递给读者。
- **抓叙事**：事件推进必须由人物主动选择驱动，禁止天降巧合和强行转折。
- **抓逻辑**：新出现的规则、能力、信息必须有铺垫；人物行为必须贴合其动机与性格。

### 三建
- **建人物**：主角必须有缺陷和成长空间；配角必须有独立动机和鲜明特征。
- **建情节**：每章都要有"目标-阻碍-行动-结果"的微型闭环，同时为长远伏笔服务。
- **建画面**：所有重要场景必须能在读者脑中形成清晰画面，具备影视改编的视觉潜力。

### 四检查
- **查悬念**：本章建立的悬念是否有信息差？揭晓时是否有足够铺垫？
- **查人物**：所有人说的话、做的事，是否符合TA此时此刻应该知道的信息和所处的情绪状态？
- **查逻辑**：时间线、地理、能力规则、人物关系是否存在矛盾？
- **查语言**：删除所有废话、套话、抽象形容词；确保句式错落、对话鲜活、用词精准。

## 四、风格化扩展关键词（新增）

### 热血磅礴风（热血玄幻）
> 世界观宏大、悬念重重、伏笔深远。人物名号响亮霸气，打斗场面要有"举世皆惊"、"震动天下"的宏大感，主角台词不屈霸气，同级无敌、逆天而行。场景描写强调"星河倒卷"、"万古长空"的史诗级画面。

### 10后向（热血中二轻小说）
> 面向 10-15 岁读者，节奏极快，三章一小爽、五章一大爽。人设标签鲜明（系统/签到/无敌流），热血中二、允许玩梗、对话活泼幽默，每章至少一个打脸/震惊全场的爽点。句子要短、段落要碎、信息密度要高。
