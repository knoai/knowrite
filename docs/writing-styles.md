# 平台风格与作者风格参考手册

> 本手册为 knowrite 小说引擎提供可直接注入 Prompt 的风格描述库。涵盖主流平台调性与各类型代表作者风格。

---

## 一、平台风格（10 种）

| 平台 | 核心特征 | 适配题材 |
|------|----------|----------|
| **番茄风** | 节奏极快，开篇 3 章内必须出现强爽点或强反转；句子短小，段落破碎，信息密度高；每 300 字一个小刺激，情绪直白浓烈，适配滑动阅读。 | 玄幻、都市、科幻、轻小说 |
| **起点风** | 允许稍长铺垫，重视世界观构建和伏笔深远；节奏张弛有度，文笔可更厚重，人物刻画更立体；支持长篇连载（百万字以上），注重 IP 改编潜力。 | 玄幻、仙侠、历史、科幻 |
| **晋江风** | 情感细腻，人物刻画立体，文笔优美；重视人物关系张力与情感递进，情节服务于人物成长与情感变化。 | 言情、耽美、仙侠、同人 |
| **七猫风** | 以甜宠或虐恋为主轴，情绪浓烈，章末钩子强；重视读者情绪共鸣，泪点/爽点密集，适合女性向快节奏消费。 | 言情、都市情感、重生复仇 |
| **飞卢风** | 脑洞大开，标题党气质，快节奏爽文；同人改编、系统流、签到流盛行；三章一小爽，五章一大爽，信息轰炸。 | 同人、系统、洪荒、都市异能 |
| **纵横风** | 玄幻仙侠大本营，文风厚重，升级体系严谨；重视战斗描写与境界突破的仪式感，热血与霸气并重。 | 玄幻、仙侠、武侠 |
| **掌阅风** | 出版向品质，结构严谨，逻辑自洽；适合影视/有声改编，重视对白质量与场景画面感，节奏稳健。 | 都市、悬疑、历史、言情 |
| **知乎风** | 短篇精悍，第一人称叙事，强反转；贴近现实，注重代入感与情绪冲击，适合中短篇或单元剧结构。 | 悬疑、现实、情感、灵异 |
| **豆瓣阅读风** | 现实主义基调，文艺气息浓厚，心理描写深入；重视社会观察与人性探讨，文笔克制而有力量。 | 现实、悬疑、女性成长、历史 |
| **刺猬猫/菠萝包风** | 二次元浓度高，玩梗密集，轻小说句式；重视人设标签、吐槽文化与 ACG 梗，宅向元素丰富。 | 轻小说、同人、二次元、系统 |

## 二、作者风格（按类型划分）

### 玄幻

1. **热血磅礴风** — 热血玄幻、霸气磅礴。世界观宏大、悬念重重、伏笔深远。人物名号响亮霸气，打斗场面要有"举世皆惊"、"震动天下"的宏大感，主角台词不屈霸气，同级无敌、逆天而行。
2. **少年逆袭风** — 退婚流/逆袭流鼻祖。少年热血、不屈不挠，修炼体系清晰，战斗节奏明快，爽点密集，情绪高昂。
3. **体系严谨风** — 设定严谨、战斗体系清晰、情感专一。擅长构建完整的世界规则与职业体系，主角坚韧正直，长线情感稳定。
4. **稳健策略风** — 番茄流代表。修炼体系自洽，节奏稳扎稳打，主角性格沉稳低调，重视家人与兄弟情，战斗讲究策略。

### 仙侠

1. **深情宿命风** — 深情仙侠、宿命哲思。擅长写"我命如妖欲封天"的逆天感，文风深情而悲壮，人物执念深重，情节中常有宿命般的无奈与不屈，意境悠远。
2. **凡人流** — 凡人流、谨慎务实。主角出身平凡，靠谨慎、毅力和算计一步步崛起。文风沉稳务实，升级体系严谨，战斗讲究策略与底牌。
3. **唯美情殇风** — 诛仙式唯美玄幻。仙侠情缘与正魔对立交织，文笔优美抒情，重视情感悲剧与人物内心刻画，意境凄美。
4. **古典正统风** — 古典仙侠正统。文化底蕴深厚，重视诗词意境与传统道家文化，画面唯美，情感含蓄。
5. **门派情义风** — 正统仙侠。重视师徒情与门派归属感，文风稳健，战斗与情感并重。
6. **脑洞仙侠风** — 开创性仙侠。脑洞大开，设定新颖，文风轻松幽默，善于打破传统仙侠套路。
7. **团战热血风** — 热血团战型仙侠。重视兄弟情义与宗门争霸，战斗场面宏大，节奏明快，热血高燃。
8. **设定控风** — 设定严谨仙侠。世界观构建极其细致，修行体系有完整理论支撑，文风偏冷静理性。
9. **家族修仙流** — 以家族/宗门整体兴衰为核心，主角不仅要个人崛起，更要带领族人共享资源、共御外敌。重视血脉传承、灵地争夺、家族议事与香火延续的集体爽感。
10. **程序员修仙流** — 用程序员的逻辑解构修仙世界：功法即代码、经脉即架构、丹药即性能优化。文风理性中透着冷幽默，爽点在于用理工科思维降维打击传统修仙者。
11. **现代农民修仙流** — 以乡土田园为根基的现代修仙。主角通过灵泉、灵植、灵田改良农作物，带动全村致富。重视乡村人情、邻里关系、田园牧歌式生活，种田与修炼并重。
12. **都市修仙流** — 主角在现代都市中修炼，身怀仙法融入繁华都市。重视扮猪吃虎、医术/武道/风水在现代社会的降维应用，以及都市人情世故与修仙者超然心态的碰撞。
13. **系统流** — 以系统面板为核心金手指。任务驱动剧情，数据化成长，完成目标即可获得奖励。重视明确的升级路径、即时反馈与确定性爽感，节奏极快。

### 都市

1. **后宫轻松风** — 都市后宫/校园保镖流。轻松搞笑，女主众多，情节轻松愉快，适合青少年读者。
2. **暧昧温情风** — 都市情感/医生文。文笔幽默，人物鲜活，擅长写暧昧与温情并存的都市恋情。
3. **灰色江湖风** — 都市黑道/异能。人物立体，剧情跌宕，擅长写灰色地带的人性与江湖义气。
4. **商战推演风** — 重生商战/官场。格局宏大，经济推演严谨，重视时代背景与商业逻辑。
5. **青春怀旧风** — 青春都市/重生怀旧。文笔细腻，重视青春遗憾与时代记忆，情感真挚动人。
6. **文娱逆袭风** — 文娱文代表。爽点密集，主角在娱乐圈/文坛一路逆袭，打脸情节设计精巧。
7. **渣男重生风** — 重生都市/渣男流。人物真实接地气，对话鲜活，擅长写青春校园与人际关系。
8. **鉴宝悬疑风** — 鉴宝/风水都市。专业知识丰富，悬疑与都市结合，重视传统文化与江湖规矩。
9. **现实热血风** — 现实主义都市。热血正义，重视社会现实与底层人物，文笔硬朗，情感浓烈。
10. **毒舌灵气风** — 灵气复苏都市。吐槽玩梗，主角毒舌，节奏快，情绪转折强烈，适合年轻读者。

### 科幻

1. **星河硬科幻风** — 硬科幻/宏大叙事。重视科学原理与哲学思辨，场面恢弘，常探讨文明存亡与人性抉择。
2. **伦理哲思风** — 哲理科幻。关注科技伦理与人类命运，情节紧凑，思想深刻，带有浓厚的人文关怀。
3. **柔情科幻风** — 软科幻/情感细腻。科学与人文并重，重视亲情与爱情在宏大背景下的脆弱与坚韧。
4. **荒诞暗黑风** — 暗黑科幻/荒诞隐喻。文风压抑诡异，擅长用科幻外壳包裹对社会现实的深刻批判。
5. **融合设定风** — 废土/赛博科幻。设定严谨，世界观融合度高，情节环环相扣，重视团队与探索。
6. **种田吐槽风** — 科幻种田/吐槽欢乐。宏大宇宙背景下的轻松叙事，玩梗密集，重视文明发展与人性温暖。
7. **硬核电影风** — 硬核科幻/电影感。科学细节考究，节奏紧凑，场景极具画面感，适合影视改编。
8. **温馨治愈风** — 科幻温馨。设定新颖但笔触温柔，重视日常生活与情感羁绊，文风治愈。
9. **机甲热血风** — 机甲科幻/热血搞笑。战斗场面精彩，主角成长线清晰，热血与幽默并存。
10. **机械策略风** — 机甲流。机械设定详细，战斗策略丰富，重视主角从弱到强的技术成长。

### 悬疑/灵异

1. **民俗探险风** — 探险悬疑/民俗丰富。重视民间传说与地理风俗，场景真实可感，悬念层层递进。
2. **盗墓解密风** — 盗墓悬疑/烧脑。挖坑与解谜并行，重视兄弟情义与家族秘密，氛围紧张诡异。
3. **都市心理惊悚风** — 心理悬疑/都市恐怖。重视心理刻画与反转，氛围阴郁，擅长写人性阴暗面。
4. **法医刑侦纪实风** — 法医刑侦/专业硬核。真实案件改编，法医知识丰富，推理过程严谨，纪实感强。
5. **犯罪心理风** — 犯罪心理/黑暗写实。重视罪犯心理画像与社会背景，文风沉重，人物命运感强烈。
6. **社会推理风** — 社会派推理。推理严密，反转惊人，重视社会现实与制度批判，结局常令人唏嘘。
7. **副本恐怖风** — 恐怖悬疑/副本精彩。以恐怖屋/冒险乐园为舞台，副本设计精巧，恐怖与温情并存。
8. **黑色灵学风** — 灵异悬疑/黑色幽默。地府、鬼神元素丰富，文风幽默中带着阴森，反套路频出。
9. **民俗阴阳风** — 民俗灵异/接地气。以阴阳先生视角展开，重视东北民俗与道教文化，语言生活化。
10. **民族神秘风** — 民族神秘学/探险悬疑。以苗疆蛊术为特色，民族风情浓厚，探险与解密结合。

### 历史

1. **穿越言情风** — 历史穿越/细腻言情。文笔优美，重视历史细节与宫廷政治，情感线缠绵悱恻。
2. **家国悲壮风** — 历史厚重/悲剧美学。重视家国情怀与乱世苍生，文风苍凉悲壮，人物命运感强烈。
3. **温情历史风** — 轻松历史/温情日常。以唐朝为背景，重视家庭温情与历史人物的再塑造，文风幽默。
4. **权谋温润风** — 权谋历史。布局深远，权斗争夺精彩纷呈，主角智谋过人，文风温润而有力。
5. **幽默历史风** — 幽默历史科普。以现代语言解读历史，幽默风趣，重视历史事件的人性化解读。
6. **谋士战略风** — 谋士流/权谋精彩。主角以智谋立足乱世，重视战略推演与朝堂博弈。
7. **军事推演风** — 硬核历史/军事推演。重视军事细节与历史推演，文风冷峻，战争场面真实残酷。
8. **民族热血风** — 热血历史/民族情怀。以南宋抗金为背景，战斗场面激昂，民族情感浓烈。
9. **变革思辨风** — 架空历史/思想深度。重视社会制度变革与思想启蒙，布局宏大，人物众多。
10. **制度严谨风** — 严谨历史/制度改革。以北宋为背景，重视政治经济改革的逻辑推演，文风沉稳。

### 言情

1. **甜宠轻松风** — 甜宠治愈。文笔温暖轻松，人物可爱，情感发展自然，结局圆满，适合放松心情。
2. **虐恋古典风** — 虐恋情深/文笔华美。情感浓烈而绝望，重视古典意境与悲剧美学，催泪效果极佳。
3. **古装虐恋风** — 穿越言情/历史厚重。以历史为背景，情感与权谋交织，人物命运波澜壮阔。
4. **酸涩青春风** — 现实言情/酸涩成长。重视青春遗憾与现实无奈，文风细腻，情感真挚。
5. **悬爱甜文风** — 悬爱言情/智商在线。将悬疑与爱情结合，推理严谨，感情线甜而不腻。
6. **耽美群像风** — 耽美言情/群像精彩。人物塑造饱满，情感浓烈，世界观完整，重视道义与情义的冲突。
7. **强强剧情风** — 强强联合/文笔老练。感情线与剧情线并重，人物独立强大，对话机锋，思想深刻。
8. **现实治愈风** — 现实向/救赎感。重视底层人物的生活细节，对话鲜活，情感温暖而真实，治愈人心。
9. **温柔浪漫风** — 温柔浪漫/设定新颖。文风温柔，情感细腻，擅长构建独特的世界观与浪漫氛围。
10. **克制遗憾风** — 现实言情/人物坚韧。重视人物内心成长，情感克制而深沉，结局常有遗憾美。

### 轻小说/二次元

1. **电竞群像风** — 电竞文/群像搞笑。人物众多但各具特色，对话幽默，重视团队协作与热血竞技。
2. **烧脑吐槽风** — 吐槽玩梗/烧脑副本。主角高智商毒舌，副本设计精巧，文风幽默中带着黑暗。
3. **毒舌灵气风** — 毒舌吐槽/灵气复苏。主角嘴贱但心怀正义，节奏快，情绪转折强烈，适合年轻读者。
4. **轻松甜文风** — 轻松搞笑/后宫甜宠。文风幽默，情节轻松愉快，女主形象鲜明，适合休闲阅读。
5. **无限战斗风** — 无限流/战斗热血。副本设计多样，战斗场面精彩，主角成长线清晰。
6. **科幻修真风** — 科幻修真/热血中二。将科幻与修真融合，世界观宏大，热血与理想主义并存。
7. **种田吐槽风** — 吐槽欢乐/科幻奇幻。宏大背景下的人文关怀，玩梗密集，重视文明与个体的关系。
8. **日常吐槽风** — 日常吐槽/无敌流。以日常生活与吐槽为主，节奏轻松，适合碎片化阅读。
9. **脑洞搞笑风** — 搞笑日常/脑洞大开。以修真聊天群为代表，打破第四面墙，玩梗无下限。
10. **苟道稳健风** — 苟道流/轻松搞笑。主角极度稳健，反差萌强烈，文风幽默，适合解压阅读。

### 年代群体风格

1. **80后风** — 热血理想与时代变迁交织。重视家国情怀、集体荣誉与奋斗精神，文风厚重，情感深沉，常带有理想主义色彩和时代记忆。
2. **90后风** — 个体觉醒与多元价值并存。重视自我表达、职场奋斗与情感独立，文风务实中带有个性张扬，善于捕捉社会热点与亚文化梗。
3. **00后风** — 极致快节奏与碎片化爽感。人设标签鲜明，玩梗密集，情绪直给，重视即时反馈与社交媒体传播性，三章一小爽，五章一大爽。
4. **10后风** — 热血中二、玩梗爽点。面向10-15岁读者，节奏极快，三章一小爽、五章一大爽。人设标签鲜明（系统/签到/无敌流），热血中二、允许玩梗、对话活泼幽默，每章至少一个打脸/震惊全场的爽点。

## 三、使用说明

- 在 **平台风格** 中选择目标发布平台的调性，AI 会自动调整节奏、句式与情绪表达方式。
- 在 **作者风格** 中选择目标风格的描述，AI 会模拟该风格的语言气质、叙事习惯与情感浓度。
- 平台风格与作者风格可组合使用（如 **番茄风 · 凡人流**），产生叠加效果。
